## SQL Associate Certification Description

This certificate confirms that the candidate has demonstrated the essential SQL skills needed for an associate-level data analyst role.

## Data Management Skills
The candidate showed the ability to clean, process, and combine data from multiple sources using SQL. They successfully completed tasks involving data quality checks, transformations, and metric calculations in PostgreSQL.

## Timed Exams
The candidate completed adaptive, timed tests covering key SQL and data concepts, showing a solid understanding of the knowledge required for the role.

## Practical Exam
Through a hands-on project in DataLab, the candidate applied their skills to solve a business problem, showing they can work with real data in a practical setting.
